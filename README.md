# library_management-system-using-docker-and-kubernate
Library Management System: Full‑stack app for managing books, members, borrow/return with auto fine calculation (₹0.50/day). Features: CRUD operations, due dates, fine tracking. Built with HTML/CSS/JS, Node.js, MongoDB. Containerized with Docker, orchestrated on Kubernetes with persistent storage, high availability, and self‑healing.
